# What is it?

Notesly is a website with **rich text notes** that can be easily **synchronized between devices via a link**.

# Quick Start

The website can be accessed at https://notesly-sync.vercel.app

# Source Code

Don't forget to change prod to dev in the _getApiLink.ts_ file (frontend/src/shared/helpers) and in the _getHostingLink.ts_ file (frontend/src/shared/helpers).

1. Clone the repository:

```
git clone https://github.com/sh4man4ik/Notesly.git
```

2. Go to the directory:

```
cd Notesly
```

3. Download frontend dependencies (from _Notesly_ directory do):

```
cd frontend
npm install
```

4. Download backend dependencies (from _Notesly_ directory do):

```
cd backend
npm install
```

5. Launch the website (from _frontend_ directory do):

```
npm run client
```

6. Launch the server (from _backend_ directory do):

```
npm run server
```

# Tech stack

The following technologies are used to develop the website: **TypeScript**, **React**, **Tailwind CSS**, **daisyUI**, **Remix Icon**, **Node.js**, **MongoDB**, **Express**, **Vitest**, **react-quill-new**, **Vite**, and **Google Fonts**.
